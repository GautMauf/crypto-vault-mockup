# Vault Risk Passport — Concept Mockup

A concept prototype exploring how Morpho Vaults could make externalized risk **legible, comparable, and monitorable** for institutional allocators.

**Thesis:** the binding constraint on the next $10B of vault deposits isn't yield or protocol capability — it's the cost of due diligence. A standardized, per-vault risk & compliance disclosure (human-readable for risk committees, machine-readable via API) collapses that cost.

## What's in the mockup

- **Passport grade** with sub-scores (guarantees / exposure / operations)
- **Noncustodial guarantees** as actually configured on-chain: timelock, in-kind redemptions, gates, exposure caps, role segregation, sentinel
- **Live exposure & concentration** with curator caps and drift alerts
- **Curator track record & audit lineage**
- **Committee view / API view toggle** — the same passport as JSON, designed for an allocator's internal risk stack
- A machine-readable **MRZ strip**, because it's a passport

## Run it

Open `index.html` in a browser — no build, no dependencies.
